发表于 2017-08-01 | 分类于 Java

介绍

HashMap是存储键-值对映射的集合，继承了AbstractMap类并实现了Map接口，可以通过put、remove、get等方法插入、更改、移除、获取键-值对映射。
HashMap内部的数据结构是哈希表，由数组、链表和红黑树组成，采用链地址法来解决哈希冲突。
